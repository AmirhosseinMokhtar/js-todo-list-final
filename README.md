# js-todo-list-final

A professional To-do list with JavaScript, CSS, HTML, this project was defined and introduced by ISACO Company. Developer: Amirhossein Rahim

In this project, I learned how to design & code with JavaScript, CSS, and HTML... And also a little bit of UI! :)
What does this project do? Adding tasks, Check in/out tasks, Deleting tasks, Filtering tasks, Counting checked tasks, Dark mode, & ...

What definitions did I use? Bootstrap CDN for UI, let / const, Arrow functions, Scope and closures, DOM Manipulation, & ...
